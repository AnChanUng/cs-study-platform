## 질문
변수에서 code 영역의 역할과, 스레드가 독립적으로 가지는 것과 공유하는 것을 설명해주세요.

## 핵심 키워드
- code 영역
- stack
- PC (Program Counter)
- 레지스터
- heap
- data

## 답변
### 변수
- **code** : 실행할 프로그램 코드

### 스레드
**스레드가 독립적으로 가지는 것:**
- stack
- PC (Program Counter)
- 레지스터

**스레드가 공유하는 것:**
- heap
- data
- code

## 꼬리 질문
1. 스레드가 stack을 독립적으로 가지는 이유는 무엇인가요?
2. heap 영역을 공유하면서 발생할 수 있는 문제는 무엇인가요?
3. PC(Program Counter)의 역할은 무엇인가요?
